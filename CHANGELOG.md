# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.3.0] - 2021-01-10

### Changed

- **sun:** shrink to balance out with other regular size icons
- use `viewBox` attribute instead of `width` and `height` for better scalability

## [0.2.0] - 2021-01-10

### Changed

- **circle-small:** shrink radius to balance out with other small icons
- **sun:** shrink to balance out with other regular size icons

## [0.1.2] - 2021-01-10

### Added

- _x_ and _x-small_ outlines
- _search_ outline
- _menu_ outline
- _minus_ outline
- _plus_ outline

## [0.1.1] - 2021-01-10

### Added

- _sun_ outline

## [0.1.0] - 2021-01-10

### Added

- base outlines:
  - _square_
  - _circle_
  - _square-small_
  - _circle-small_
- rules to follow when authoring icons
